## Installation

1. Clone the repository:

```
git clone [https://github.com/hamdyhaggag/Pet-Shop-UI.git])
```

2. Navigate to the project directory:

```
cd Pet-Shop-UI
```

3. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```
